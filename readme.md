  

  
# TinyWebDB_SAE_PHP (TSP)
在新浪云(SAE)上，跑Appinventor(或thunkable)的TinyWebDB(网络微数据库)   
语言：PHP + HTML + JavaScript( + CSS)

特点：

- 独立呗   
  不是每个人都喜欢使用公用数据库嘛
- 便宜啊   
  费用压到新浪云最低价格了    
  一天一毛钱，一年也才￥36.5好伐，，   
- 然后是有一些特色小功能   
  比如：标签的前缀计数、批量获取、搜索标签   
  数据库备份/恢复   
  附带文件空间以及其管理系统(位于后台)   
  大量实用后台管理功能
